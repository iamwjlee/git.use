1. install vsc
2. install git
3. vsc reopn for git

4. make projects folder and then code .
5. git clone https://github.com/iamwjlee/git.use
6. goto git.use directory
7. for commit updated files you need belows
    git config --list
    git config --global user.name "wj"
    git config --global user.email "iamwjlee@gmail.com"
    git config --list

8. comment first and then commit
9. git push https://github.com/iamwjlee/git.use
